## Run Locally

Clone the project

```bash
  git clone https://github.com/ak3311g/Takeleap.git
```

Go to the project directory

```bash
  cd Takeleap
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
